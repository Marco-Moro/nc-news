GET /api

Serves a JSON representation of all the available endpoints of the API. 

{
  "endpoints": {
    "/api": "API overview",
    "/api/topics": "List of topics",
    "/api/articles": "List of articles",
    "/api/articles/:article_id/comments": "Add a comment"
  }
}

GET /api/topics

Serves an array of all topics.

{
  "topics": [
    { "slug": "football", "description": "Footie!" }
  ]
}

GET /api/articles

Serves an array of all articles, supports optional queries: sort_by, order and topic. 

{
  "articles": [
    {
      "title": "Seafood substitutions are increasing",
      "topic": "cooking",
      "author": "weegembump",
      "created_at": "2018-05-30T15:59:13.341Z",
      "votes": 0,
      "comment_count": 6
    }
  ]
}

GET /api/users

Serves an array of all users.

{
  "users": [
    {
      "username": "butter_bridge",
      "name": "jonny",
      "avatar_url": "https://www.healthytherapies.com/wp-content/uploads/2016/06/Lime3.jpg"
    }
  ]
}

GET /api/articles/:article_id

Serves an article object by article_id, including the total count of comments. 

{
  "article": {
    "author": "butter_bridge",
    "title": "Living in the shadow of a great man",
    "article_id": 1,
    "body": "I find this existence challenging",
    "topic": "mitch",
    "created_at": "2020-07-09T20:11:00.000Z",
    "votes": 100,
    "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700",
    "comment_count": 11
  }
}

GET /api/articles/:article_id/comments

Serves an array of comments for the given article_id, sorted by most recent first. 

{
  "comments": [
    {
      "comment_id": 1,
      "votes": 16,
      "created_at": "2022-09-09T20:12:00.000Z",
      "author": "butter_bridge",
      "body": "Oh, I've got compassion running out of my nose, pal! I'm the Sultan of Sentiment!",
      "article_id": 1
    }
  ]
}

POST /api/articles/:article_id/comments

Adds a new comment to the article with the specified article_id. 

Request Body:
{
  "username": "butter_bridge",
  "body": "Oh, I've got compassion running out of my nose, pal! I'm the Sultan of Sentiment!"
}

Example Response:
{
  "comment": {
    "comment_id": 19,
    "votes": 0,
    "created_at": "2025-06-08T10:00:00.000Z",
    "author": "butter_bridge",
    "body": "Oh, I've got compassion running out of my nose, pal! I'm the Sultan of Sentiment!",
    "article_id": 1
  }
}

PATCH /api/articles/:article_id

Updates the vote count of the specified article.

Request Body:
{
  "inc_votes": 1
}

Example Response:
{
  "article": {
    "article_id": 1,
    "title": "Living in the shadow of a great man",
    "topic": "mitch",
    "author": "butter_bridge",
    "body": "I find this existence challenging",
    "created_at": "2020-07-09T20:11:00.000Z",
    "votes": 101,
    "article_img_url": "https://images.pexels.com/photos/158651/news-newsletter-newspaper-information-158651.jpeg?w=700&h=700"
  }
}

DELETE /api/comments/:comment_id

Deletes the comment with the specified comment_id.
