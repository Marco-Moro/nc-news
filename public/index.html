Endpoints

  • GET /api/topics
  • GET /api/articles
  • GET /api/users

GET /api/topics

Restituisce tutti i topic disponibili.

{
  "topics": [
    {
      "slug": "coding",
      "description": "Code is love, code is life"
    }
  ]
}

GET /api/articles

Restituisce tutti gli articoli. Accetta le query sort_by, order e topic.

{
  "articles": [
    {
      "article_id": 1,
      "title": "Running a Node App",
      "author": "jessjelly"
    }
  ]
}

GET /api/users

Restituisce tutti gli utenti.

{
  "users": [
    {
      "username": "butter_bridge",
      "name": "Jonny"
    }
  ]
}
